Modules Implemented Details

1

Data Acquisition & Harmonization

Retrieved clinical datasets from Cleveland (303 instances) and Statlog (270 instances). Applied feature harmonization to ensure matching cross-domain attributes and schema consistency.

2

Data Preprocessing

Performed rigorous missing value imputation, robust outlier removal (IQR methodology), one-hot encoding for categorical variables, standard scaling, and SMOTE-ENN class balancing.

3

Model Training & Tuning

Trained XGBoost, LightGBM, and Random Forest base learners utilizing Optuna for Bayesian hyperparameter optimization. Evaluated using 5-fold cross-validation on the source domain.

4

Stacking Ensemble Creation

Integrated base learners into a meta-classifier architecture using a Logistic Regression meta-learner (C=0.1) to intelligently weigh and combine base predictions for superior accuracy.

5

Cross-Domain Evaluation

Deployed the source-trained (Cleveland) ensemble directly onto the unseen target domain (Statlog) to record the model's accuracy degradation and real-world generalizability.

6

SHAP Explainability & Feature Shift

Extracted model feature importance via SHAP values for both domains. Computed attribution shifts to understand how diagnostic reasoning changes across varying patient populations.

Result Discussion & Feature Shift

Our findings reveal an underlying vulnerability in clinical machine learning: Attribution Shift. Even when overall predictive accuracy remains acceptable, the internal logic the model uses changes drastically depending on the population demographics.

Comparative SHAP Summary Plots — SHAP values visually rank the importance of features. Comparing these domains demonstrates that the model relies heavily on 'thalach' and 'age' in the source domain, but the prioritization shifts considerably on the target Statlog population.
Feature Distribution Shift — Highlights statistical variations in patient data distributions (such as cholesterol ranges or age curves) between datasets, which directly causes the model's accuracy drop.
